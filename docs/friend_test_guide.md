# 朋友测试说明

感谢你帮忙测试 LoL BP Coach。这个版本主要测试 BP 阶段的识别、推荐和战术分析是否真的能用。

## 一句话说明

这是一个英雄联盟 BP 辅助工具：进入 BP 后，它会识别双方阵容，然后给出英雄推荐、对线推荐、路线强弱和战术建议。

## 怎么开始

1. 安装 `LoL-BP-Coach-Setup-0.2.0.exe`
2. 打开 `LoL BP Coach`
3. 选择你要打的位置
4. 点击 `启动识别`
5. 进入 LOL BP 界面
6. 等待软件自动刷新推荐

如果暂时没有开游戏，可以点 `演示阵容` 看完整效果。

## 主要按钮

- `启动识别`：开始读取 BP 界面。
- `停止识别`：停止后台识别。
- `定格`：锁定当前推荐，防止后续误识别覆盖。
- `继续`：解除定格，继续自动刷新。
- `新局`：清空当前结果，准备下一局。
- `演示阵容`：载入一套测试阵容。

## 你要重点看什么

- 推荐英雄是否符合你的位置？
- 推荐英雄是否明显离谱？
- 对线推荐是否有优势逻辑？
- 战术建议是否能看懂？
- 路线强弱是否符合阵容直觉？
- UI 有没有挡住、错位、太小、看不清？

## 如果出问题

请把下面信息发回来：

- 问题截图
- 你选的位置
- 双方阵容
- 软件显示的推荐
- 是否点过 `定格` 或 `新局`

## 注意

这个版本还不是最终版，主要用来找真实 BP 场景里的问题。推荐结果仅供参考，不保证游戏胜负。
