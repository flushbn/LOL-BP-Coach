# LoL BP Coach v0.2.0 朋友测试版

这是 LoL BP Coach 的第一个朋友测试安装包版本。

本版本重点不是“最终成品”，而是验证：下载安装后是否能直接打开、进入 BP 后是否能识别阵容、推荐和战术分析是否能稳定刷新。

## 推荐下载

优先下载：

- `LoL-BP-Coach-Setup-0.2.0.exe`

如果安装包被安全软件拦截，备用下载：

- `LoL-BP-Coach-portable-0.2.0.zip`

便携包解压后直接运行里面的启动程序即可。

## 本版本包含

- PySide6 单窗口桌面客户端
- LOL BP 截图识别流程
- 综合英雄推荐
- 对线推荐
- 路线强弱分析
- 战术建议与节奏计划
- 定格 / 继续 / 新局功能
- 我的数据与手动历史胜率录入
- 16.13 版本数据
- 中文英雄名与英雄头像资源
- Lolalytics 缓存和种子缓存

## 基本使用流程

1. 安装并打开软件
2. 在左上角选择自己的分路
3. 点击 `启动识别`
4. 打开英雄联盟并进入 BP 界面
5. 等待推荐、对线和战术模块自动刷新
6. 阵容稳定后点击 `定格`
7. 下一局点击 `新局`

## 已知限制

- 当前仍处于朋友测试阶段，识别速度和准确率还会继续优化。
- LOL 客户端必须保持可见，不要最小化。
- 如果 BP 界面被其他窗口挡住，识别可能会变慢或失败。
- 推荐结果是辅助决策，不保证胜负。
- 部分协同、战术和节奏判断属于本地推断数据。

## 希望朋友反馈什么

请重点反馈以下问题：

1. 软件能不能正常打开？
2. 进入 BP 后有没有刷新识别结果？
3. 英雄推荐是否看起来合理？
4. 对线推荐是否符合实际经验？
5. 战术建议有没有帮助？
6. 哪些界面看不懂或太挤？
7. 有没有闪退、卡死、无数据？

反馈时最好附带：

- 截图
- 自己选择的位置
- 双方阵容
- 当时软件显示的推荐结果
