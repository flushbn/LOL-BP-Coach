# LoL BP Coach

LoL BP Coach 是一个面向《英雄联盟》BP 阶段的 Windows 桌面辅助工具。它通过截图识别客户端 BP 信息，在 PySide6 单窗口中提供英雄推荐、对线分析、阵容战术与个人数据面板。

> 当前发布版本：`v0.3.6`  
> 当前数据版本：`16.14`（Riot 官方公告版本：`26.14`）

## 核心功能

- **BP 识别**：读取可见的 LOL BP 客户端画面，识别双方英雄、Ban 位和目标分路。
- **英雄推荐**：综合版本强度、分路适配、Counter、协同、阵容需求、对线优势与玩家熟练度。
- **对线推荐**：展示当前分路下更具对位优势的选择，并显示位置推断。
- **战术教练**：分析双方阵容的前排、开团、保护、爆发、持续输出与后期能力，给出简明战术建议。
- **版本强势**：按上路、打野、中路、下路、辅助展示主流分路 Top10 胜率英雄；额外展示高胜率非常规选择，并明确与主流榜分区。
- **英雄详情**：点击英雄卡查看版本数据、出装路线、阵容适配与对线思路。
- **我的数据**：记录胜负、导入历史胜率，形成英雄池与熟练度画像，并影响推荐结果。
- **版本公告**：展示 Riot 官方版本公告、英雄/装备改动及对推荐的解释。
- **局内战术浮窗**：显示已定格的阵容与战术信息；不读取游戏内存、不注入进程、不自动操作键鼠。

## 安装与使用

1. 前往 [Releases](https://github.com/flushbn/LOL-BP-Coach/releases) 下载最新安装包。
2. 运行 `LoL-BP-Coach-Setup-0.3.6.exe`，选择安装位置。
3. 启动 **LoL BP Coach**，先选择自己的分路。
4. 打开英雄联盟客户端并进入 BP 界面。
5. 点击“启动识别”，等待界面刷新推荐。
6. BP 结束前可点击“定格”锁定结果；下一局点击“新局”清空当前草稿。

没有 BP 场景时，可点击“演示阵容”预览完整 UI。

## 数据与可信度

- **Lolalytics GLOBAL / Emerald+ / Ranked Solo-Duo**：版本强度、分路胜率与对位数据。不是腾讯国服专属统计。
- **Riot Data Dragon**：英雄中文名、基础资料与头像资源。
- **Riot 官方公告**：版本改动解释与版本影响。
- **本地推断**：阵容协同、路线节奏和部分 Coach 建议；界面会将推断数据与实证数据区分。

版本强势页的主流榜会过滤低样本、低选取率和非主流分路英雄；例如法师下路会进入“非常规英雄”区域，而不会与传统 ADC 榜混合。

## 数据更新

客户端会检查新 Patch。若 Lolalytics 直接请求受到 Cloudflare 限制，软件会保留已验证的旧数据，避免产生空推荐。

维护端使用 Firecrawl 抓取并校验新版本数据后，可发布新的数据包。普通用户应从 GitHub Release 下载已验证的安装包或数据更新包，而不需要直接抓取 Lolalytics。

```

python build_release.py
```

## 注意事项

- LOL 客户端需要保持可见，不能最小化。
- 推荐仅作为 BP 决策参考，不保证对局胜负。
- 项目与 Riot Games 无官方关联。League of Legends、英雄名称和相关素材版权归 Riot Games 所有。
