# LoL BP Coach

LoL BP Coach 是一个面向《英雄联盟》BP 阶段的桌面辅助工具。它通过截图识别 LOL 客户端 BP 信息，并在 PySide6 单窗口客户端中展示英雄推荐、对线推荐、阵容分析、路线强弱、装备建议和玩家数据。

当前版本定位：朋友测试版。目标是尽量做到“打开客户端 → 进入 BP → 查看推荐 → 定格结果 → 赛后记录”的完整流程。

## 最新进度

- 已统一为 PySide6 单窗口 UI，不再依赖旧 Overlay 展示结果。
- 已接入 BP 截图识别，并将识别结果写入 `data/live_state.json` 供 UI 刷新。
- 已完成综合推荐、对线推荐、战术分析、路线强弱、已选英雄、我的数据、版本更新、数据更新中心等页面。
- 已选英雄页支持手动搜索添加英雄，适合玩家自己选择已锁定英雄后查看装备。
- 点击英雄卡片会打开英雄详情，当前重点展示装备推荐，不再展示符文栏。
- 装备推荐优先读取本地 Lolalytics 缓存；点击“更新全英雄在线数据（含装备）”会一起刷新装备数据。
- 推荐页支持定格 / 继续 / 新局，避免 BP 界面变化导致识别误刷新。
- 新增局内战术浮窗，安全展示 BP 阶段已生成的路线强弱、阵容对比和战术建议。

## 核心功能

### BP 识别

- 点击 `启动识别` 后，程序读取 LOL BP 界面截图。
- 识别己方英雄、敌方英雄、Ban 位和当前分路。
- UI 会显示识别状态，方便判断是否已经刷新。

### 综合推荐

综合推荐会结合以下因素：

- 当前版本强度
- 分路适配度
- 敌方英雄克制关系
- 己方阵容需求
- 团队协同
- 对线优势 Lane Bonus
- 玩家熟练度 Comfort Bonus

推荐卡片显示英雄头像、综合分、核心标签和快速装备提示。

### 对线推荐

- 基于 Lolalytics matchup / delta 数据。
- 根据当前分路和敌方可能对位英雄，显示有对线优势的英雄。
- 不强制放满 10 个，只展示更有价值的优势选择。

### 已选英雄

已选英雄页用于查看“玩家已经选到 / 准备玩的英雄”。

- 支持自动读取己方已选英雄。
- 支持手动搜索添加英雄。
- 每个英雄会单独推断分路，不会再把所有英雄都当作当前顶部按钮分路。
- 后台加载装备推荐，不拖慢综合推荐页。
- 点击卡片可进入英雄详情页查看完整装备路线。

### 英雄详情 / 装备推荐

点击推荐英雄、对线英雄或已选英雄卡片后，会打开英雄详情。

当前详情重点展示：

- 当前版本数据
- 核心三件套
- 出门装
- 第 4 / 5 / 6 件候选装备
- 阵容适配装备
- 对线思路
- 强势期分析
- 克制 / 协同参考

装备数据来源优先级：

1. `data/cache/lolalytics/{patch}/builds/` 本地在线缓存
2. 英雄专属快照
3. 本地定位兜底装备

### 战术教练

战术模块包含：

- 己方阵容分析
- 敌方阵容分析
- 双方对比
- 前排 / 开团 / 保护 / 爆发 / 持续输出 / 后期能力
- 路线强弱分析
- 打野行动建议
- 中期资源与节奏建议

### 局内战术浮窗

局内浮窗用于进入游戏后查看已经生成的静态战术卡片：

- 显示阵容快照、核心节奏、路线强弱、阵容对比和战术建议。
- 支持置顶、拖动、滚动、暂停刷新、手动刷新、折叠、透明度调节和位置保存。
- 默认使用精简模式，只显示主节奏、主攻路、防守路、关键路线和最多 3 条建议；也可以切回完整模式。
- 默认优先显示主客户端 `定格` 后的 BP 结果，避免进入游戏后识别误刷新影响阅读。
- 只读取 `data/live_state.json`，不读取游戏内存，不注入游戏进程，不自动操作鼠标键盘。
- 可从主客户端点击 `局内浮窗` 打开，也可以运行 `python run_ingame_overlay.py` 单独打开。

### 我的数据

我的数据页用于建立玩家画像：

- 记录本局胜负
- 导入使用软件前的历史胜率
- 统计英雄场次、胜率、最近使用
- 识别核心英雄和谨慎选择英雄
- 后续会影响推荐中的熟练度加成

### 数据更新中心

数据更新中心用于软件内更新游戏数据：

- 检查当前 patch
- 切换 patch
- 一键更新数据
- 更新全英雄在线数据（含装备）
- 重建缓存
- 查看更新日志

装备推荐缓存会随“更新全英雄在线数据（含装备）”一起更新。

## 使用方式

### 普通测试用户

推荐使用 Release 安装包：

1. 下载 Release 中的安装包或便携包。
2. 解压或安装后双击启动。
3. 打开 LOL 客户端并进入 BP。
4. 在 LoL BP Coach 中选择分路。
5. 点击 `启动识别`。
6. 推荐稳定后点击 `定格`。
7. 下一局点击 `新局`。

如果暂时没有 BP 场景，可以点击 `演示阵容` 查看完整 UI 效果。

### 开发者运行

```powershell
pip install -r requirements.txt
python run_app.py
```

如果只想打开新版 UI：

```powershell
python run_ui.py
```

构建发布包：

```powershell
python build_release.py
```

## 数据来源

- **Lolalytics**：版本强度、对线 matchup、英雄构筑、装备路线、缓存数据。
- **Riot Data Dragon**：英雄中文名、基础数据、头像资源。
- **本地推断数据**：阵容协同、路线节奏、战术建议、部分 Coach 规则。

部分战术和协同判断属于本地推断，不等同于官方或职业数据。

## 注意事项

- LOL 客户端需要保持可见，不要最小化。
- 当前仍是测试版，推荐结果用于 BP 参考，不保证胜负。
- 如果识别错位，可先点击 `定格` 保留当前正确结果。
- 如果装备或英雄数据过旧，请到 `数据更新中心` 执行更新。
- 本项目与 Riot Games 无官方关系。
